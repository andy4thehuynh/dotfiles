# DOTFILES
![](https://i.imgur.com/kRXMMrX.png)

## Setup
1. Clone the repo `$ git clone git@github.com:andy4thehuynh/dotfiles.git`
2. cd into the repo and create the symlinks by executing `make bootstrap`
3. Verify symlinks i.g. navigate to `~/.config` and it should return the symlink if you run `readlink nvim`

### Ensure you set your terminal to a Nerdfont for icons
1. Navigate to [NF downloads](https://www.nerdfonts.com/font-downloads)
2. Select, download and install font to your Font Book (macOS)
3. Ensure your terminal emulator settings is using the font

### Raycast Setup
1. Create API keys for Mistral, ChatGPT and Gemini
2. Open Raycast extensions and set the keys to the appropriate LLM
