# Mac

```
# install homebrew
$ /bin/bash \
-c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# install packages, casks (apps), taps (repos)
$ brew bundle Brewfile

# downloads software organizer
$ brew install stow

# symlink all files
$ stow .
```
